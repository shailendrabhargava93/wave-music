## PWA Features

This app is a Progressive Web App, which means:

- **Installable**: Add the app to your home screen on mobile or desktop
- **Offline Support**: Previously played songs and cached content work offline
- **Fast Loading**: Assets are cached for instant loading
- **Auto-updates**: App automatically updates when new versions are available
- **Native Feel**: Runs in standalone mode without browser chrome

### Installing the App

#### On Mobile (Android/iOS):
1. Open the app in your browser (Chrome, Safari, etc.)
2. Look for the "Add to Home Screen" or "Install" prompt
3. Tap "Install" or "Add"
4. The app will appear on your home screen like a native app

#### On Desktop:
1. Open the app in Chrome, Edge, or another supported browser
2. Look for the install icon in the address bar (âŠ•)
3. Click "Install"
4. The app will open in its own window
